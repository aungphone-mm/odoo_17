Telegram Bot Integration

Easily integrate Telegram with Odoo for automated, real-time notifications sent directly to users via a Telegram Bot.

Category

Extra Tools

Summary

Integrate Telegram Bot for automated message notifications in Odoo.

Description

This module provides seamless integration between Telegram and Odoo, enabling users to receive real-time notifications, updates, and alerts directly through a Telegram Bot. Enhance your Odoo environment's communication capabilities by establishing a direct channel for timely and effective user notifications.

Features

  • Automated notifications and alerts sent directly to Telegram.
  • Improves communication by providing real-time updates.
  • Easy setup and configuration to link Odoo with a Telegram Bot.

Usage

After installing this module, configure your Telegram Bot token in Odoo settings. Once configured, you can start sending automated notifications to users through Telegram.

Configuration

Follow these steps to configure the integration:

Create a Telegram Bot and get the API token. You can refer to the official Telegram documentation for more information on creating a bot and obtaining the API token. https://core.telegram.org/bots/tutorial
  1. Go to Odoo General Settings and enter your Telegram Bot API token.
  2. Fetch Telegram users which already connected with your bot.
  1. Go to the user form and select the Telegram user to receive notifications.
  1. Now, when there is a user notification odoo will send notifications to the selected Telegram user.
